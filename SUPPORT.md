# Support

AppShelfKit is an early open-source Swift Package for indie Apple developers.

## Where to Get Help

- Use GitHub Issues for reproducible bugs.
- Use GitHub Discussions if the repository enables them.
- Use pull requests for focused fixes or documentation improvements.

## Before Opening an Issue

Please check:

- the README
- DocC documentation
- existing issues
- the sample app in `Examples/AppShelfKitExample/`

Include the platform, Xcode version, Swift version, package version, and a small reproduction when possible.

## Support Boundaries

Maintainers can help with AppShelfKit behavior and documentation. Maintainers cannot provide guaranteed support for:

- custom backend hosting
- App Store Connect configuration
- third-party image CDNs
- private app codebases
- unofficial App Store APIs

For 0.x releases, the package is usable but still evolving.
